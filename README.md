# Physics Kruting Hub

Physics Kruting Hub เป็นแพลตฟอร์มการเรียนรู้ฟิสิกส์ระดับชั้นมัธยมศึกษาปีที่ 6 (เล่ม 5) ที่พัฒนาขึ้นเพื่อช่วยให้นักเรียนสามารถเรียนรู้เนื้อหาฟิสิกส์ผ่านเว็บไซต์เชิงโต้ตอบ (Interactive Learning Web Apps)

โครงการนี้ครอบคลุมเนื้อหาตามหลักสูตร สสวท. โดยเน้นการเรียนรู้ผ่านบทเรียนดิจิทัล อินโฟกราฟิก แบบจำลองเชิงโต้ตอบ แบบฝึกหัด และแบบทดสอบออนไลน์

### Interactive Learning Experience

* บทเรียนแบบ Interactive
* Infographic สรุปเนื้อหา

* สรุปบทเรียนแบบเข้าใจง่าย

* ติดตามความก้าวหน้าการเรียน
* บันทึกคะแนนแบบทดสอบ
* ระบบ Achievement
* Learning Analytics

### Modern Web Technology

* Responsive Design
* Progressive Web App (PWA)
* Mobile Friendly
* Fast Loading
* Offline Support (บางส่วน)

## Installation

Clone Project

```bash
git clone https://github.com/nopphorn31/physics-kruting-hub.git
```

Enter Project Directory

```bash
cd physics-kruting-hub
```

Run Local Server

```bash
npx serve
```

หรือเปิดผ่าน Live Server ใน VS Code
